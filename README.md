Smart Text Analyzer – A Cloud-Based Web Application

📖 Description

The Smart Text Analyzer is a web-based application developed using Python and Flask. It allows users to input any paragraph of text and instantly analyzes it to provide useful insights such as word count, character count, vowel count, longest word, and more.
This project is deployed on a cloud platform and does not require any database, making it lightweight and efficient.

🎯 Objectives

To build a simple cloud-based application
To analyze text using Python logic
To implement frontend and backend integration
To deploy the application on a cloud platform

🛠️ Technologies Used

Python
Flask
HTML
CSS
JavaScript
Cloud Platform (Render / AWS / Azure)

✨ Features

Word Count
Character Count
Vowel Count
Longest Word Detection
Text Reversal
Uppercase & Lowercase Conversion
Top 3 Frequent Words
Palindrome Word Detection
Responsive and Modern UI
Dark Mode Toggle
